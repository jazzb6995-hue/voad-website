Setup Guide

Three Things to Set Up

This guide walks you through the three setup steps: creating your Google Sheet (your CMS), connecting it to the website, and uploading to GoDaddy. Once done, you add or remove projects simply by editing a spreadsheet — no code needed.

A

Create Your Google Sheet (the CMS)

1

Create a new Google Sheet

Go to sheets.google.com and create a new blank spreadsheet. Name it VOAD Projects.

2

Set up the Projects sheet — paste these column headers in Row 1

Click the tab at the bottom named Sheet1 and rename it to Projects. Then paste these exact headers into row 1 (one per cell, A1 through P1):

id title category tagline shortDesc longDesc year location area scope cover gallery1 gallery2 gallery3 gallery4 featured

Each column explained:

ColumnWhat to put hereExample
idUnique slug, no spacesmehra-residence
titleProject nameThe Mehra Residence
categoryMust be one of these exactly: residential / interior / heritage-commercialresidential
taglineOne-line subtitle shown on the detail pageA family home built around light
shortDesc2 sentence summary shown on cardsA 4,200 sq ft bungalow…
longDescFull description for the project detail page. Use blank lines to separate paragraphs.The brief was…
yearCompletion year2024
locationCity, StateRajkot, Gujarat
areaFloor area (optional)4,200 sq ft
scopeServices deliveredArchitecture · Interior Design · Turnkey
coverFull URL of the cover/hero imagehttps://…
gallery1–4Full URLs of up to 4 gallery imageshttps://…
featuredShow on homepage? Type true or falsetrue
Tip: Add one row per project. The website reads every row automatically. To add a project — add a row. To delete — delete the row. To update — edit the cell.
3

Where to host your project photos

Upload your real project photos to Google Drive. Right-click each photo → Share → Anyone with the link can view. Then copy the sharing URL and use it in the cover / gallery1–gallery4 columns.

Better option: Upload images directly to GoDaddy inside a folder called images/projects/ and use relative URLs like images/projects/mehra-residence-01.jpg. This is faster and more reliable than Google Drive links.
B

Connect the Sheet to Your Website

4

Open the Apps Script editor

In your Google Sheet, click Extensions → Apps Script. A new tab opens with a code editor.

5

Paste the script code

Delete everything already in the editor. Open the file google-apps-script.js from your website folder, copy its entire contents, and paste it into the Apps Script editor. Click Save (disk icon).

6

Deploy as a Web App

Click Deploy → New Deployment. Configure it exactly like this:

Type: Web App Description: VOAD CMS Execute as: Me (your Google account) Who has access: Anyone

Click Deploy. Google will ask you to authorise — click through and allow. You will see a URL ending in /exec. Copy that URL.

7

Paste the URL into config.js

Open js/config.js in your website folder. It looks like this:

const VOAD_CONFIG = { appsScriptUrl: '', // ← paste your URL here liveFromSheets: false // ← change to true };

Paste the /exec URL between the quotes. Then change false to true. Save the file. The website will now load projects directly from your Google Sheet.

Test it first: Before uploading to GoDaddy, open index.html in your browser. Your real project data should appear within a second or two. If it doesn't, double-check the URL and that "Who has access" is set to Anyone.
8

Test the contact form

Fill in the contact form on contact.html and submit it. Wait 10 seconds, then check your Google Sheet — a new tab named Contact should appear automatically with your test submission.

Note: Because the form uses a privacy-safe "no-cors" request, the browser can't confirm receipt — but the data is always written. The "Thank you" message shows immediately after submission.
C

Deploy to GoDaddy

9

Log in to GoDaddy cPanel

Go to your GoDaddy account → My Products → Hosting → Manage. Click cPanel Admin to open the hosting control panel.

10

Open File Manager

In cPanel, find and click File Manager. Navigate to the public_html folder — this is the root of your website. Everything you upload here will be live at your domain.

11

Upload your files

Select all files and folders from your voad-website folder on your computer:

index.html portfolio.html about.html contact.html project.html setup-guide.html ← optional, can delete after setup google-apps-script.js ← optional, can delete after setup css/ js/

In File Manager, click Upload and drag these in. Make sure the folder structure is preserved — css/ and js/ must remain as subfolders.

Tip: You can also use FTP (FileZilla) for faster bulk uploads. GoDaddy provides FTP credentials under cPanel → FTP Accounts.
12

Visit your domain

Open your domain in a browser. The VOAD website should load. All navigation, project pages, filter, and contact form should work immediately. If images are missing, check that your image URLs are correct.

✓

Day-to-Day: Adding & Removing Projects

Add a new project

Open your Google Sheet → Projects tab. Add a new row at the bottom with all the columns filled in. The project appears on your website within seconds — no upload needed.

Remove a project

In the Projects sheet, right-click the row and select Delete row. It disappears from the website immediately.

Edit a project

Find the row in the Projects sheet and edit any cell. Changes are live as soon as you save the sheet.

Change featured projects

The homepage shows the first project per category where featured = true. Change a project's featured column to swap what shows on the homepage.

View contact submissions

Every form submission is saved to the Contact tab in your Google Sheet automatically. You can also get email notifications by going to Tools → Notification rules in Sheets.

Swap placeholder images

Upload real project photos to your GoDaddy images/projects/ folder, then update the cover and gallery1–gallery4 columns in the Sheet with the new URLs.

You're all set.

Your website is a fully working CMS-driven portfolio. Projects are managed from Google Sheets, contact submissions go directly to your Drive, and the whole thing runs on GoDaddy static hosting with no monthly CMS fees. When you're ready to swap placeholder images for real project photos, simply update the image URLs in the Sheet.
